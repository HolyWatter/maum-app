## 채팅이 가능한 데스크탑 앱

### 기간 : 1월 10일 ~ 1월 14일 (약 5일)

### 사용기술
nextron + react + typescript + firebase

styled with tailwind css

state manage : Recoil

### 기능
- [x] firebase auth를 활용한 회원가입 및 로그인
- [x] 유저목록 불러오기
- [x] firebase firestore를 활용한 실시간 1:1 채팅기능
- [x] 그룹채팅기능
